## Tip para Linux

Mueve el CSV a una ruta simple para evitar problemas de permisos:

```bash
# En la terminal de Linux
mv deibis.csv /tmp/deibis.csv
```

Luego en DBeaver:

```sql
COPY invitados(id_invitado, PRIMER_NOMBRE, SEGUNDO_NOMBRE, PRIMER_APELLIDO, SEGUNDO_APELLIDO, TELEFONO, IDENTIFICACION, GENERO)
FROM '/tmp/deibis.csv'
DELIMITER ';'
CSV HEADER;
```

> `/tmp/` es la carpeta más fácil en Linux porque PostgreSQL siempre tiene permisos para leerla */
